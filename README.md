# About this mod

This is a minecraft mod that adds a OP omnitool.
This tool can harvest almost every block in the game.
Has several modes:
 - 3x3 mode; 
 - silk & fortune mode;
 - vein miner mode for ores;
 - extended vein miner for any other blocks.

# State

This mod is pretty much complete.
